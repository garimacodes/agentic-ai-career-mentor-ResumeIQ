#  Agentic AI Career Assistant

An AI-powered career guidance platform that helps users analyze resumes, practice mock interviews, and receive personalized feedback using Large Language Models.

---

##  Overview

The **Agentic AI Career Assistant** is designed to solve a common problem faced by students and job seekers — lack of structured career guidance and interview preparation.

This platform acts as a **virtual AI coach** that:
- Reviews resumes
- Conducts mock interviews
- Evaluates answers in real-time
- Provides actionable insights

---

##  Features

###  Resume Analysis
- Upload resume (PDF)
- Extracts and analyzes content
- Provides strengths, weaknesses, and improvement suggestions

###  Mock Interview System
- Generates role-based interview questions
- Supports multiple difficulty levels (Easy, Medium, Hard)
- Mix of Technical, Behavioral, Situational, and Domain questions

###  AI Feedback Engine
- Evaluates answers using LLM
- Provides:
  - Score (out of 10)
  - Strengths & weaknesses
  - Sample strong answer
  - Improvement tips

###  Voice Input
- Answer questions using speech (Web Speech API)
- Converts voice to text in real-time

###  Camera-Based Interview Simulation
- Enables webcam during interview
- Creates a realistic interview environment

###  Performance Summary
- Generates overall interview report
- Includes score, confidence level, and recommendations
